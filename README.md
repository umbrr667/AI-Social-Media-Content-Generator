# AI-Social-Media-Content-Generator
#AI-powered Social Media Content Generator built with Python and Groq's Llama 3.3 model. Generate platform-specific content calendars, captions, hashtags, and marketing posts for LinkedIn, Instagram, and X (Twitter).

#🤖 AI Social Media Content Generator
📌 Overview

The AI Social Media Content Generator is a Python application that uses Groq's Llama 3.3 70B model to generate engaging social media content for different platforms.

Users simply enter a product or brand name, provide a short description, choose a platform, and specify the number of days for a content calendar. The application generates platform-specific marketing content and saves it automatically.

✨ Features
📢 Generate marketing content using AI
📅 Create multi-day content calendars
💼 LinkedIn content generation
📷 Instagram captions
🐦 X (Twitter) posts
🏷️ AI-generated hashtags
💾 Save generated content automatically
⚡ Powered by Groq Llama 3.3 70B

Python
Groq API
Llama 3.3 70B Versatile
